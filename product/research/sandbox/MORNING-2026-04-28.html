3架构亮点 · 第一性原理设计

领域模型纯净

core/skill_script.py 只有 pydantic + stdlib · 零 I/O · ScriptInvocation 是 LLM ↔ runner 契约 · ScriptResult 是 runner ↔ tool 契约 · interpreter 决议是它自己的方法。

ABC 解耦执行

ScriptRunner Protocol 让 SubprocessScriptRunner / FakeScriptRunner / 未来 ContainerScriptRunner / RemoteScriptRunner 可热插拔 · tool 层只依赖接口。

沙盒边界清晰

safe_resolve_script(scripts/ 前缀强制 · symlink 拒)+ env 白名单(防泄密)+ timeout(防卡死)+ RLIMIT_AS(防爆内存)+ stdout 截断(防 token 爆)。

ADR 0021 自解释

错误回 {error, field, expected, received, hint} · LLM 看完自纠 · 不吐 stack trace · tool description 用陈述句不用 "MUST"。

5.1 跑 builtin script_demo skill(无需任何 API key)

cd backend
uv run python -c "
import asyncio
from datetime import UTC, datetime
from allhands.core import Employee, SkillRuntime
from allhands.execution.script_runner import SubprocessScriptRunner
from allhands.execution.skills import SkillRegistry, seed_skills
from allhands.execution.tools.meta.resolve_skill import make_resolve_skill_executor
from allhands.execution.tools.meta.skill_scripts import make_run_skill_script_executor

async def main():
    reg = SkillRegistry(); seed_skills(reg)
    runtime = SkillRuntime()
    emp = Employee(id='e', name='t', description='d', system_prompt='sp',
                   model_ref='p/m', skill_ids=['allhands.script_demo'],
                   tool_ids=[], created_by='u', created_at=datetime.now(UTC))

    await make_resolve_skill_executor(employee=emp, runtime=runtime,
                                       skill_registry=reg)(skill_id='allhands.script_demo')

    out = await make_run_skill_script_executor(
        runtime=runtime, skill_registry=reg, runner=SubprocessScriptRunner()
    )(skill_id='allhands.script_demo', script='scripts/echo.py',
       args=['hello', 'morning'])
    print('✓ exit:', out['exit_code'], '· stdout:', out['stdout'].strip())

asyncio.run(main())
"
