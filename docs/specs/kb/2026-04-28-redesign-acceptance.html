5 · 已知 follow-up · 不阻塞验收

实测过程顺手记下来的,不是这次范围,但你知道我看到了。

  • reindex 重置 counter 健壮性
    reindex_document 失败时 bump_counters(chunks=-N) · N 是被删的 chunk 数 · 但 KB.chunk_count 已经 0 时仍减 → 负数 · 再失败 reindex 累积。修法:bump 改成 absolute set(read real count from kb_chunks)· 或 reindex 末尾再 reconcile 一次。已 live-fix · 留 ticket。
  • 小文档不 chunk
    R4 测试用的 1 行 md · upload 完 chunks=0(state=ready)· 不报错但没东西可搜。Chunker 应该至少出 1 个 chunk · 或拒绝过短输入并提示。
  • PDF parser 缺依赖
    实测发现 pypdf 没装 · 一上传 PDF 就 failed。pyproject.toml 缺 dep · 加上即可。
  • Ask thread 持久化
    现在 Ask 多轮存在 React state · 刷新就丢。Conversation 持久化是单独 feature(需要新表)· 不在本次范围。
  • Settings/Diagnose stats 在多 KB 下需要按 KB 隔离
    现在内存 ring buffer 是全局 · 切 KB 还看到上一个的统计。小细节 · 后期改 dict-by-kb-id 即可。
