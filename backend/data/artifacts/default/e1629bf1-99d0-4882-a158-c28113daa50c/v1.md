# 制品区能力测试员 - 能力全景图

## 1. 多模态文档生成
- **Markdown / Code**: 基础文本与代码片段管理。
- **PDF**: 正式报告、长文打印输出。
- **Word (docx)**: 结构化提案、协议、合同。
- **PPT (pptx)**: 演示文稿、季度汇报。

## 2. 数据与表格处理
- **Excel (xlsx)**: 多工作簿复杂数据分析。
- **CSV**: 标准化数据导出，兼容 CJK 编码。

## 3. 可视化与图表
- **Draw.io**: 流程图、架构图、时序图、ER 图。
- **Mermaid**: 快速关系链、思维导图渲染。
- **HTML**: 交互式仪表盘、单页 Demo。
- **Image**: 图像资产存储与预览。

## 4. 全生命周期管理
- **版本控制**: 支持迭代更新、回滚至历史版本。
- **检索归档**: 全文搜索、分类筛选、置顶管理。
- **清理机制**: 软删除保护，防止误操作。
